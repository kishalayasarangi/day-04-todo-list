# Day 04 — To-Do List App

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A terminal-based to-do list app in Python. Tasks are saved
to a JSON file so they persist even after you close the app.

## How to run
```bash
python3 todo.py
```

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Tasks saved to JSON (persists between sessions)
- Clean terminal UI
